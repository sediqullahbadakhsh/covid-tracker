## Setup

### Clone this repository

```bash
$ git clone git@github.com:sediqullahbadakhsh/covid-tracker.git
$ cd covid-tracker
```

### Run project

```bash
$ npm install
$ npm run start # this will make webpack watching for your changes in code
```
